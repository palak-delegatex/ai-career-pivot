# AICareerPivot

Your personalized AI-powered career transition strategist.

## What It Does

AICareerPivot builds custom career roadmaps by analyzing your:
- Current skills and experience
- Financial runway and constraints
- Family responsibilities and risk tolerance

It produces actionable **6-month**, **1-year**, and **2-year** transition plans — not generic advice.

## Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **AI**: [Anthropic Claude](https://anthropic.com) via Claude Agent SDK
- **Deployment**: [Vercel](https://vercel.com)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the app.
